About Me

PASSION PROJECT - Zach's Museum

My life-long passion for learning about history inspired me to create this. Using the Cleveland Museum of Art's massive 37,000+ online collection, I created a website where a user can select one of the museum's many departments, as well as an artwork make/type, and view a carousel scroll of up to ten completely random items from that department's collection of art masterpieces. This project utilized HTML, CSS, Sass, and Vanilla Javascript.

Juno College Project 2: Event Ticket Finder - Sport Seeker

This project was my first experience with paired programming. Along with my partner Jay Park, we used Ticketmaster's public API to fully design and build a responsive website which allows the user to pick their current city and preferred team sport. Upon form submission, the user is returned a comprehensive list of all upcoming sporting events in their area, along with a link to purchase tickets. This project is fully responsive and utilizes HTML, CSS, Sass, and Vanilla Javascript.
